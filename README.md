# Teste - Biblioteca

## Exercícios de criação de de biblioteca visual (Não completo)

## Utilização

### build

Na raiz do projeto

```s
mvn clean install
```

### Deploy

Na raiz do projeto

```s
mvn spring-boot:run
```

## Métodos para acesso via url do browser.

### Utilizadores

- http://localhost:8081/users

### Livros

- http://localhost:8081/books/available
